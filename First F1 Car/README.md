# 🏎️ Formula One Car in SolidWorks

## 🎯 About this Project
This project documents my journey of **designing and assembling a Formula One car** in SolidWorks as I work towards my dream of becoming a motorsports engineer. It is more than just a CAD project. It's an **engineering portfolio** that captures my learning process, design decisions, and technical growth. Each modeling session includes progress updates, reflections, and lessons learned, allowing me to showcase not only the finished model but also the **problem-solving and engineering mindset** developed throughout the project. While the project is guided by reference tutorials, I continually **incorporate my own ideas and design choices** so that my confidence and understanding naturally grow.
